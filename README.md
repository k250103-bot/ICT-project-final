
# **Calorie Calculator Web App**

A simple and interactive web application that calculates **maintenance**, **bulking**, and **cutting** calories based on user inputs such as **age**, **height**, **weight**, **gender**, and **activity level**.
Created as an **ICT project** using HTML, CSS, and JavaScript.
